# 🍽️ FoodHub - Restaurant E-commerce Website

A full-stack restaurant e-commerce web application developed as part of the **CodeAlpha Full Stack Development Internship**.

FoodHub provides a seamless online food ordering experience with a modern Angular frontend, secure REST APIs, and a PostgreSQL database.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Zainab090/CodeAlpha_FoodHub.git
```

### 2. Navigate to the Project

```bash
cd MY-RESTAURANT
```

### 3. Install Frontend Dependencies

```bash
cd frontend
npm install
```

### 4. Install Backend Dependencies

```bash
cd ../backend
npm install
```

### 5. Configure Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
PORT=5000

DB_HOST=localhost
DB_PORT=5432
DB_USER=your_username
DB_PASSWORD=your_password
DB_NAME=foodhub
```

---

## ▶️ Run the Project

### Start Backend

```bash
npm start
```

or

```bash
npm run dev
```

### Start Frontend

```bash
ng serve
```

Open your browser:

```
http://localhost:4200
```
